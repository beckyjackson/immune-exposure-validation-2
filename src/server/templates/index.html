Immune Exposure Model

An immune exposure is the process by which components of the immune system first encounter a potential trigger. This model applies FAIR principles to the description of the events, materials, diseases, and specimens relevant to immune response, facilitating accurate capture and retrieval of immune exposure records for scientific studies and databases.

On this site you fill find:

  • controlled terminology to capture:
    • subject ID
    • event type
    • start and end time
    • exposure material type
    • disease type
    • disease stage
    • specimen type
    • specimen ID
  • instructions for using the model
  • examples of the model
  • convenient Excel and CSV templates
  • web-based validator for filled templates

This prototype builds on earlier work in A structured model for immune exposures which is used by ImmPort for its "Immune Exposure" template (see ImmPort templates). The new version supports a time-series of relevant life events and adds support for specimens.
